
# GSE268686_RNA_Analysis Report
This report analyses two QuantSeq 3' mRNA-seq experiments from the GEO dataset GSE288686, each targeting a distinct histone-modifying enzyme in Arabidopsis thaliana via CRISPR/Cas9. The experimental design — with guide RNA knockout lines, no-guide controls, and an rdr6 epigenetic mutant background — mirrors the logic of CRISPR/Cas9 editing validation used in chromatin engineering projects such as NOR relocation in Draba huetii.
